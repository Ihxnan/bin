# 个人命令行工具集合

实用 shell 脚本和可执行文件，用于系统管理、进程控制、显示配置等。

## 工具列表

| 工具 | 功能 |
|------|------|
| `dp` | C++ 程序自动化测试脚本 |
| `key` | 交换 Caps Lock 和 Ctrl 键 |
| `move` | 批量移动文件（Python） |
| `mykill` | 根据进程名终止进程 |
| `pipes` | 终端动画屏保 |
| `toggle-boot-target` | 切换系统启动目标（图形/命令行） |
| `tuo` | 配置双显示器（HDMI 右侧扩展） |
| `u` | 挂载/卸载 `/dev/sda1` 设备 |
| `yun` | 云服务器管理（SSH/挂载/文件传输） |
| `zhu` | 配置主显示器（HDMI 镜像） |

## 使用

```bash
chmod +x <脚本名>
./<脚本名> [选项]
```

## 环境要求

- Linux（Arch/Manjaro）
- Bash
- g++、Python 3、xrandr、sshfs、variety
